Cloud Geospatial Data Visualization Platform

Welcome! Visualize your COG, Zarr and GeoParquet data stored in the cloud. Enjoy a simple and fast interface to explore large geospatial datasets, anytime and anywhere

Get Started

Visualise cloud-based geospatial data such as Zarr, Cog, and Geoparquet in the web

This project is a web platform designed to visualize cloud-based geospatial data formats, including Zarr, COG (Cloud Optimized GeoTIFF), and GeoParquet.

The platform features interactive web maps capable of rendering large-scale geospatial datasets directly, without the need to convert the data to GeoJSON format.

Cloud data visualization is crucial for accessing and analyzing vast datasets quickly and efficiently. This platform addresses these challenges by providing:

  • A user-friendly interface that enables visitors to explore three sample datasets (Zarr, COG, and GeoParquet).
  • Interactive maps that dynamically display and navigate the different data formats.
